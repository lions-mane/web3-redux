Interface BlockHeader

Block header object. Typically returned by Web3 websocket subscriptions. Extends the web3 interface.

param

Block id.

param

A network id.

param

Number: The block number. null if a pending block.

param

32 Bytes - String: Hash of the block. null if a pending block.

param

32 Bytes - String: Hash of the parent block.

param

8 Bytes - String: Hash of the generated proof-of-work. null if a pending block.

param

32 Bytes - String: SHA3 of the uncles data in the block.

param

256 Bytes - String: The bloom filter for the logs of the block. null if a pending block.

param

32 Bytes - String: The root of the transaction trie of the block.

param

32 Bytes - String: The root of the final state trie of the block.

param

32 Bytes - String: The root of the final receipt trie of the block.

param

String: The address of the beneficiary to whom the mining rewards were given.

param

String: The “extra data” field of this block.

param

Number: The maximum gas allowed in this block.

param

Number: The total used gas by all transactions in this block.

param

Number: The unix timestamp for when the block was collated.
